搭建准备

创建GITHUB账号

首先我们需要在Github上注册账号。打开github官网点击右上角的”Sign Up”

填写对应信息进行注册即可。提交注册收需要邮箱验证，点击邮箱中github发送的邮件的验证按钮，即可完成验证。

在Github创建用于图床的仓库

点击右上角的加号按钮，然后在弹出的菜单中选择”new repository”来创建新的仓库

在Repository name文本框中填入仓库名称，在Description中填入简介，选择Public(公开的仓库)然后点击Create repository来创建仓库。

接下来获取github 提供的api的key。点击右上角头像的按钮。点击后在后续菜单中选择Settings。

在出现的页面的侧边栏中找到Developer Settings选项

然后点击左侧的”Personal access tokens”菜单，在新出的页面中点击”Generate new token”

在Note中填写介绍，Select Scopes下面的所有选项全部选择。

最后点击”Generate token”创建token key。记得把token key保存好。因为只会出现一次。该key最后会用于PicGo的Github图床。

在Github创建应用于Pages服务的仓库

同样创建新的仓库。在仓库名称中填写”github用户名.github.io”。格式是固定的，不要填错。仓库类型同样选择public。在Initialize this repository with的初始化可以选择Add a README file(这一步可以不做，创建了README后可以方便测试)。然后点击Create repository创建该仓库。之后会跳转到仓库页面。选择上面的Settings来设置该仓库。在该页面中找到GitHub Pages这一栏会看到一个提示”Your site is published at https://用户名.github.io”就说明创建成功了。如果之前选择了add a readme file的话可以访问这个链接，就可以看到README文件中的内容。

GIT工具下载地址。下载后双击安装，除了路径之外，其他的选项默认即可。

安装完成后需要通过git的命令行进行配置。在开始菜单中找到Git->Git Bash点击后就可以打开git的命令行。执行两条命令。

然后我们打开GitHub官网登录后进行公钥的配置。点击用户头像选择Settings在侧边的菜单中点击SSH and GPG keys。点击 New SSH Key。

在弹出的窗口的Title中填写标题（比如MyPc）然后在key中填写刚刚生成并复制的公钥文本。然后点击”Add SSH Key”。

安装PicGo

首先去PicGo官网下载PicGo软件下载地址

之后按照步骤一步步安装即可。安装完毕后我们将在PicGo中配置我们之前创建的GITHUB图床仓库。

打开安装完毕后的PicGo。在侧边的图床设置中我们找到gitHub图床

然后再仓库名称中设置”github用户名/仓库名称”，分支名填写master，并将之前之前创建图床仓库时设定的token key复制到设定Token中。然后点击设为默认图床然后点击确定即可。

安装Typora

安装很简单，下载后按照步骤安装即可

下载地址

然后我们对typora简单配置一下。

点击文件->偏好设置->图像。

插入图片时我们选择上传图片。上传服务设定里选择PicGo(app)。然后再PicGo的路径中选择picgo.exe文件所在路径即可。

至此准备工作就算完成了。
